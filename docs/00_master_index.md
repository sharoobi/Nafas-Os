What this project is:

- A personal, local-first anti-smoking operating layer
- A predictive context engine, not a generic quit counter
- A hybrid Flutter shell with native Android/iOS capability engines
- A configurable system with an internal lab and experiment console

What this project is not:

- A generic motivation app
- A server-first health platform
- A hidden surveillance app built around covert camera usage
- A medical diagnosis product

North-star interaction:

`The app should know when the user is likely to smoke, intervene before the act, and provide a fast sensory/behavioral alternative that is easier to accept than reaching for a cigarette.`

Non-negotiable technical constraints:

- Offline-first
- Local-only by default
- No backend in v1
- Progressive permissions
- Native platform bridges for high-value sensors
- Rules-first, ML-second

Non-negotiable UX constraints:

- No shame-driven copy
- No noisy alert spam
- Rescue actions must be one or two taps max
- Interventions must feel calming, not punitive

Delivery phases:

1. Strategic docs
2. Project scaffold
3. Core shell and design system
4. Logging and rescue flows
5. Context engine and rules
6. Insights and lab
7. Guarded audio and advanced intelligence
